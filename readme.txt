Die Webseite https://www.handy-flatrates.com/ nutzt Content-Egg um die Datensätze der großen Mobilfunlk-Anbieter in das System zu laden. Leider unterstützt diese Software in Deutschland nur AWIN und Tradedoubler, alle anderen Datensätzen aus anderen Netzwerken müssen manuell geladen werden. Hier sollen daher die Mapping Informationen gesammlt werden um die Datensätze andere Anbieter per Content Egg in eine Wordpress Umgebung zu laden.
