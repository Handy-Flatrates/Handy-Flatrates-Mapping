Die Webseite https://www.handy-flatrates.com/ nutzt Content-Egg um die Datensätze der großen Mobilfunlk-Anbieter in das System zu laden. Leider unterstützt diese Software in Deutschland nur AWIN und Tradedoubler, alle anderen Datensätzen aus anderen Netzwerken müssen manuell geladen werden. Hier sollen daher die Mapping Informationen gesammlt werden um die Datensätze andere Anbieter per Content Egg in eine Wordpress Umgebung zu laden. Diese Daten sind auch für andere Webseiten nutzbar, die dieses Plugin einsetzen, denn die Zuordnung von API DatenFeldern zu internen Datenfeldern ist bei jedem Einsatz des Plugins gleich.

Projekt Home: https://www.handy-flatrates.com
Projekt Besipielseite: https://www.handy-flatrates.com/9859/xiaomi-14-mit-handy-flat-im-ueberblick/

Das Content-Egg Wordpress Plugin ist in der Free-Version hier zu finden: https://de.wordpress.org/plugins/content-egg/ Die Details zum Fiel Mapping bei noch nicht unterstützten Feeds gibt es hier: https://ce-docs.keywordrush.com/modules/feed-modules/field-mapping
