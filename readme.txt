Die Webseite https://mobile-dealz.de/ nutzt Content-Egg um die Datensätze der großen Mobilfunk-Anbieter in das System zu laden. Leider unterstützt diese Software in Deutschland nur AWIN und Tradedoubler, alle anderen Datensätzen aus anderen Netzwerken müssen manuell geladen werden. Hier sollen daher die Mapping Informationen gesammlt werden um die Datensätze andere Anbieter per Content Egg in eine Wordpress Umgebung zu laden. Diese Daten sind auch für andere Webseiten nutzbar, die dieses Plugin einsetzen, denn die Zuordnung von API DatenFeldern zu internen Datenfeldern ist bei jedem Einsatz des Plugins gleich.

Projekt Home: https://mobile-dealz.de/
Projekt Besipielseite: https://mobile-dealz.de//9859/xiaomi-14-mit-handy-flat-im-ueberblick/

UPDATE Das Projekt wurde im Sommer 2024 umgezogen. Man findet die Dateien und die Webseite nun unter neuer URL, die Beschreibung wurde entsprechend aktualisiert.

Das Content-Egg Wordpress Plugin ist in der Free-Version hier zu finden: https://de.wordpress.org/plugins/content-egg/ Die Details zum Field Mapping bei noch nicht unterstützten Feeds gibt es hier: https://ce-docs.keywordrush.com/modules/feed-modules/field-mapping
